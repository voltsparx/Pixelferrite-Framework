# Console Commands

Planned command groups:
- global: `help`, `banner`, `version`, `clear`, `history`, `exit`
- tools policy: `tools list`, `tools allow <tool>`, `tools deny <tool>`, `tools reset`
- scope: `scope list`, `scope add`, `scope remove`, `scope enforce on|off`, `scope clear`
- module: `search`, `show payloads|exploits|auxiliary|encoders|evasion|nops`, `use`, `info`, `set`, `unset`, `run`, `check`, `back`
- workflow: `embed`, `extract`, `mutate`, `simulate`, `analyze`
- session/log: `sessions`, `log show`, `report generate`
- external tools: `nmap <args>`, `netprobe-rs <args>` (must be installed and available in PATH)
